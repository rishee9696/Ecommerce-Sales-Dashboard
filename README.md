# Ecommerce-Sales-Dashboard

E-Commerce Sales Dashboard | Power BI & Excel

An interactive E-Commerce Sales Dashboard built using Power BI and Microsoft Excel to analyze sales performance, profit trends, customer behavior, product categories, payment methods, and regional sales distribution. The dashboard provides actionable insights through dynamic visualizations, KPI cards, slicers, and trend analysis to support data-driven business decisions.

Purpose
Monitor overall sales, profit, quantity sold, and average performance metrics.
Analyze sales and profit trends across different months and states.
Identify top-performing customers, categories, and sub-categories.
Evaluate customer payment preferences and purchasing patterns.
Enable stakeholders to make informed decisions through interactive filtering and visualization.


🛠️ Tech Stack

📊 Power BI – Interactive dashboard creation and data visualization
📑 Microsoft Excel – Data cleaning, preparation, and analysis
⚡ Power Query – ETL (Extract, Transform, Load) and data transformation
📈 DAX (Data Analysis Expressions) – Custom measures, KPIs, and calculated fields
🗄️ Data Modeling – Building relationships and optimizing data structures
📉 Business Intelligence (BI) – Turning raw data into actionable insights

📂 Data Source

🛒 E-Commerce Sales Dataset
📄 Source Format: Microsoft Excel (.xlsx)
📊 Data Includes: Orders, Sales Amount, Profit, Quantity, Customer Details, Product Categories, Sub-Categories, Payment Modes, States, and Order Dates
Source:- Chat GPT.

✨ Features & Highlights

📊 Interactive Power BI dashboard with dynamic filters
📈 Real-time tracking of Sales, Profit, Quantity, and Average Order Value
🗺️ State-wise sales performance analysis
📅 Monthly profit trend visualization
👥 Customer purchasing behavior insights
🛍️ Category and sub-category performance analysis

🎯 Actionable insights for data-driven business decisions
